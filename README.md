### Requirements
- Git 2.54 or higher
- Python 3.x or higher
- [VideoSubFinder](https://www.videohelp.com/download/VideoSubFinder_6.10_x64.zip?r=fcBBfmZrTz) 6.10 or higher

### Run
```bash
git clone https://github.com/lionc2240/autovsf.git
cd autovsf
pip install watchdog google-api-python-client oauth2client httplib2 opencv-python psutil Pillow
python .\main.py
```
